Environment
==

## Install

```
$ git clone このリポジトリ
$ cd ./environment
$ chmod +x setup.sh
$ ./setup.sh
```

## Directory Structure

```
.
├── README.md
├── bin/           バイナリ
├── dotfiles/      ドット(.)から始まるファイル
│   ├── shell/     - シェル設定
│   ├── bashrc     - bash設定
│   ├── emacs.d/   - emacs設定
│   └── gitconfig  - git設定
├── lib/           ライブラリ
└── setup.sh       セットアップスクリプト
```
