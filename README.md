Environment
==

## Install

```
$ git clone このリポジトリ
$ cd ./environment
$ chmod +x setup.sh
$ ./setup.sh
```

## Directories

```
.

├── bin/           バイナリ
├── dotfiles/      ドット(.)から始まるファイル
```
